# 产品范围

## 项目简介

Spring 生态学习项目是一个系统化的 Spring 生态学习资料库，包含完整的中英对照文档和独立可运行的示例项目，覆盖数据库集成、微服务架构、框架核心能力三大板块。

## 项目定位

与零散的博客文章或视频教程不同，本项目提供：

- **结构化学习路径**：从核心基础到微服务进阶的推荐学习路线
- **独立可运行的代码**：每个示例项目都是一个完整的 Spring Boot 应用，可直接运行
- **中英双语文档**：同时面向中文开发者和国际开发者
- **Docker 一键环境**：所有中间件通过 Docker Compose 统一管理，零本地安装负担

## 目标

- 为 Java/Spring 开发者提供一站式学习 Spring Boot + Spring Cloud 全生态的资源
- 通过独立可运行的示例降低学习和复现门槛
- 建立从入门到微服务架构的系统化学习路径
- 沉淀 Spring 生态最佳实践和常见坑点

## 目标用户

- **初级 Java 开发者**：通过核心基础篇快速上手 Spring Boot
- **中级 Java 开发者**：通过数据库/框架篇深入 Spring Data、Security 等模块
- **高级/架构师**：通过微服务篇掌握 Spring Cloud 分布式架构
- **技术面试准备者**：系统回顾 Spring 生态各模块核心知识点

## 功能需求

### 文档体系

- 按主题组织（核心基础 / 数据库 / 框架核心 / 微服务 / 进阶主题）
- 每个主题多个子文档，每个文档深度覆盖一个技术点
- 中英对照，技术术语保持英文
- 包含配置示例、代码片段和架构图

### 示例项目

- 每个示例项目为独立可运行的 Spring Boot 项目
- 使用 Maven 构建，Java 21 + Spring Boot 4.0.5
- 配套 Docker Compose 文件启动所需中间件
- 代码风格统一，结构清晰

### 学习路径

- 提供推荐学习顺序
- 模块化设计，可按需跳转学习

## 非功能性需求

- **性能**：文档渲染流畅，示例项目启动时间合理
- **安全**：示例代码不包含真实密钥和凭证
- **兼容性**：兼容 macOS / Linux / Windows 开发环境
- **国际化**：文档支持中文和英文

## 入口模式

- 文档网站（Markdown 渲染）
- Maven 命令行运行示例项目
- Docker Compose 管理中间件环境

## MVP 范围

### MVP 包含

- 核心基础篇文档（IoC, MVC, AutoConfig, Transaction）
- 数据库篇文档（MySQL, Redis, MongoDB）
- 框架核心篇文档（Security, AOP, Actuator）
- 微服务篇文档（Gateway, Discovery, Config）
- 每个文档对应的独立示例项目
- Docker Compose 中间件编排

### MVP 不包含

- 微服务高级主题（分布式事务、可观测性）— 文档已有，示例项目延后
- 进阶数据库（ES, ClickHouse, InfluxDB）— 文档已有，示例项目延后
- 进阶主题（WebFlux, Batch, Modulith）— 文档已有，示例项目延后
- CI/CD 流水线自动化
- Web 端在线交互式学习平台

### MVP 约束

- 文档优先于示例项目，部分示例项目标记为"待生成"
- 示例项目目标版本 Spring Boot 4.0.5 + Java 21，当前部分暂为 3.5.0 + Java 17

## 验收标准

- 每个完成的文档包含不低于 500 字的核心内容
- 每个完成的示例项目可独立编译运行
- Docker Compose 可一键启动所有中间件
- 推荐学习路径可串联所有模块

## 延期项

- Spring Cloud Alibaba 全家桶（Sentinel, Seata, RocketMQ）集成示例
- Kubernetes 部署方案
- 在线交互式代码编辑环境
- 视频教程配套
